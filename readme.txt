== Description ==
*Para usar este Plugin, debe estar dado de alta en [TiendaCrypto](https://tiendacrypto.com)*

Con el Plugin para WooCommerce de TiendaCrypto, vas a poder cobrar con criptomonedas de la forma más sencilla. 

La integración es rápida y gratuita. Tus cobros los recibís al instante y la comisión es muchísimo menor que en los pagos con tarjeta. Además, podés aceptar las redes más populares: BSC y Ethereum. El proceso es simple: luego de instalarlo y cuando tu cliente vaya a pagar, lo único que tiene que hacer es seleccionar la opción de TiendaCrypto Commerce y pagar con su wallet preferida (TiendaCrypto, Binance, Metamask, Trust Wallet, etc).

Con los cobros que recibas, tenés dos opciones:
1- Conservarlos en cripto.
2- Pasar las criptos a pesos argentinos automáticamente y sean depositados en tu CBU/CVU. 

Integra la API Commerce de TiendaCrypto de la manera más fácil y marcá la diferencia ofreciendo cripto como medio de pago a tus clientes.

Para darte de alta seguí esta guía:
[Guía para darse de alta en Commerce](https://tiendacrypto.notion.site/Alta-en-TiendaCrypto-Commerce-bcd496f0d0b2425e8df10aec879a2ccb)

== Installation ==

El plugin se mantiene actualizado para ser compatible con la última versión de WooCommerce y su instalación es bastante simple:

1) Instalá y activá el plugin TiendaCrypto para WooCommerce en tu tienda.

2) Ingresá a la sección “WooCommerce → AJUSTES" del menú WooCommerce, en el panel izquierdo de WordPress.

3) Acede a la pestaña “PAGOS”.

4) Dentro de "PAGOS" vas a poder ver el método de pago TiendaCrypto. Ingresar al mismo.

5) Cargar la información que el plugin solicita. *TC API KEY y WEBHOOK SECRET KEY son datos que te va a dar TiendaCrypto.

6) Todo listo!
